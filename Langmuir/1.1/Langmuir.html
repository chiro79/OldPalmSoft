Langmuir

This program returns the value of Langmuirs for a given pressure and a given exposure time. 1 Langmuir is defined as the exposure of gas corresponding to 1.0e-6 torr per second.

This program is FREE. Two are the available versions: PalmOS and MS Windows.

V. 1.0 - PalmOS (3.x - 4.x - 5.x)

Requirements: MathLib library (included in the package) or available here.
