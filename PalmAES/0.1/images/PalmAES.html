PalmAES  0.1 (alpha) (for PalmOS 3.x - 4.x - 5.x)

This program provides a reference for Auger Electron Spectroscopy (AES). More information about AES can be found here. This is an alpha version, in early stage of development. The database, and the GUI is incomplete for the most part. This version is supposed to be a preview of the future releases. The database is collected from here:

Handbook of Auger Electron Spectroscopy (2nd edition)

L.E.Davis, N.C.MacDonald, P.W.Palmberg, G.E.Riach, R.E.Weber

Physical Electronics Division

Perkin-Elmer Corporation
         

Since this is supposed to be a scientific oriented application, any comments for improvements of this application will be very appreciated. 

This program is FREE.
