PalmTVA 1.5  PalmOS (3.5 - 4.x - 5.x - 6.x)

English version of this page

Calculatrice pour le Taxe sur la Valeur Ajoutée. Saisi le montant TTC ou HT et sélectionner le taux de TVA, 5,50 % au 19,90 %.
Si vous souhaitez annuler et de recommencer, il vous suffit de cliquer sur le bouton effacer. Ce logiciel devient vite indispensable, même pour un usage personnel.

Freeware (gratuit)

Téléchargez

PalmTVA.zip (6Ko)

PalmTVA.prc (14Ko)

Informations d’Installation

Téléchargez le fichier: PalmTVA.zip depuis la page web et décompressez le sur votre PC.
Installez PalmTVA.prc sur votre Palm en activant la synchronisation classique (hotsync).
Si vous souhaitez installer une nouvelle version, vous n'êtes pas obligé de supprimer la précédente de votre PDA.
