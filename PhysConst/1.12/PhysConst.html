PhysConst  1.12 (for PalmOS 3.x - 4.x - 5.x/Garnet - 6.x/Cobalt)

This program provides a collection of physics constants. Every constant is provided in different units. A list of the fundamental and derived SI units is also provided. All the constant can be copied in the clipboard to be used in other programs.

This program is FREE.
