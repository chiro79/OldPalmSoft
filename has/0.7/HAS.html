Program for computing Helium beam energies, wavelength and momentum and beam velocity from Helium Atom source temperature. For Helium Atom Scattering users.

This program is FREE. 

Requirements:

Mathlib library needs to be installed. A copy of it is provided with the package.

Download:

ZIP File (26K)
